# weather_forecast
**simple weather_forecast with reactjs**  
___
API data is provide by [OpenWeather](https://openweathermap.org/api)

## Requirements
node packet: [nodejs](https://nodejs.org/en/)

## Usage
run `npm install` to install essential packet, after that run `npm start`
```
npm install

npm start
```

Open localhost:3000 and take a look around
